# Dash & Slash Demo

A responsive side-scroller prototype showcasing:
- WASD movement
- Flip animations
- Dash with cooldown and speed boost
- Unity Input System

## Controls
- Move: A/D or ← →
- Dash: Left Shift

## Built With
- Unity 2022+
- Input System
- Coroutines
